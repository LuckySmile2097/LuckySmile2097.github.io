基于GitHub Pages和Hexo搭建个人博客

摘要：

本篇文章主要讲述了笔者如何使用GitHub Pages和hexo搭建自己的个人博客，当然其中主要也是参考了前辈们的教程才顺利完成的，当中也碰到了很多问题，下面是本人的创建过程。

一、系统环境配置

要使用Hexo，首先系统中要支持Nodejs和Git，所以第一步就是安装这两个程序。

安装Node.js：

  • 下载并安装Node.js
  • 在后面安装hexo的时候出现了问题，刚开始安装的64位的无法使用，后面换成32位的就成功了。

    安装Git

  • 下载地址：点击此处
  • 安装教程：windows下安装Git
  • Git的基本操作：常用基本命令

    二、安装Hexo

    Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
  • 安装好Git之后，回到桌面单击右键，选择Git Bash Here，如图：

  • 然后输入安装Hexo的命令：

    npm install -g hexo-cli

安装成功之后会出现这条语句：

  • 之后我们便可以选择一个硬盘目录作为文件的存放路径，本人是在E盘下建立了一个blog文件夹，然后在Git中执行以下命令完成hexo的初始化：

    hexo init e:blog

  • 再进入blog文件夹所在的目录：

    cd e:\blog

  • 执行以下命令，系统会根据package.json文件中dependencies的配置安装所以依赖包：

    npm install

  • 然后生成部署文件，启动本地服务：

    hexo g
hexo s

更多Hexo的命令用法可以参考Hexo官网。

hexo会默认生成一个HelloWorld的博文，打开http://localhost:4000/就可以看到我们刚才搭建的本地博客了。

三、GitHub Pages设置

1、为什么使用GitHub

  • GitHub是基于git实现的代码托管。git可能是目前最好用的版本控制系统了，非常受欢迎。
  • GitHub可以免费使用，并且快速稳定。
  • Github上面的世界很精彩，用久了你的眼界会开阔很多。

    2、什么是GitHub Pages？

    Github Pages可以被认为是用户编写的、托管在github上的静态网页。GitHub Pages本用于介绍托管在GitHub的项目，但是由于它的空间免费稳定，使用Github Pages可以为你提供一个免费的服务器，所以非常适合用来搭建博客。

    3、注册GitHub账号

    在创建博客之前，当然必须有GitHub的帐号，该帐号将用来创建项目。进入GitHub官网，需要注意的是邮箱十分重要，GitHub上很多通知都是通过邮箱的。比如你的主页上传并构建成功会通过邮箱通知，更重要的是，如果构建失败的话也会在邮件中说明原因。

    4、创建项目仓库

    注册账号登录之后，点击右上角的加号点击new repository，注意库名必须是username.github.io

    5、配置SSH

  • 上传文件需要配置ssh key，不然无法上传。
    首先先检查一下本地是否已经存在ssh key,在Git Bash输入以下指令（任意位置点击鼠标右键），检查是否已经存在了SSH keys。

    ls -al ~/.ssh

    如果不存在就没有关系，如果存在的话，直接删除.ssh文件夹里面所有文件：

  • 之后设置name和email：

    git config --global user.name "<your name>"
git config --global user.email "<your email>"

  • 生成SSH密钥：输入以下指令后回车

    ssh-keygen -t rsa -C "XXXXXX@qq.com"

    一直回车之后，再输入指令：

    cat ~/.ssh/id_rsa.pub

    然后把生成的密钥复制下来

  • 在GitHub上添加SSH密钥：在https://github.com/settings/keys中点击add new ssh key

title可以随意取个名字，GitHub上的工作就完成了。

四、部署到GitHub

配置_config.yml

  • 编辑blog文件夹下的_config.yml，修改Deployment部分

    # Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
type: git
  repo: ssh://git@github.com/LuckySmile2097/LuckySmile2097.github.io
  branch: master

  • 安装Git包，执行以下命令

    npm install hexo-deployer-git --save

  • 部署到GitHub上，执行以下命令

    hexo deploy

    然后就可以通过访问http://LuckySmile2097.github.io/来访问自己的博客了。

  • 如果要更改主题，只需将主题下载，然后移到blog的themes文件夹下，再把E”\blog_config.yml里的theme名字改为自己下载的主题的文件夹名就可以了。

  • 这样我们的博客就创建好了。
